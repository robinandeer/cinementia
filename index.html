  • What is this?

    MonoFra.me is a movie memory guessing game. You're asked to identify a movie from a single frame. Simple as that.

  • Local

    The game stores all progress locally in your browser. It is pretty permanent but for now, you can't bring your results with you between computers (unless you're tech savvy and know what HTML5 localStorage is). You can however very easily compare highscores with anyone who played the game on a single computer!

  • Features

    The current lack of features is all part of the plan...to get the site up and running...and to avoid responsibility for your data. But hey, highscore and basic leveling already works!

  • Creator

    Robin Andeer, a biotech student in denial. For some time, I've wanted to create a proof-of-concept web project around movies and a game finally made the most sense. Big thanks to Google App Engine for free web hosting. You can find more about me here: www.robinandeer.com
